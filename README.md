Инструкция по сборке и запуску проекта:

Требования
* Java 17
* Maven 3.x
* Среда разработки (IntelliJ IDEA, VSCode и др.)

Шаги по сборке и запуску

1. Клонировать репозиторий или скачать исходный код.
2. Перейти в каталог проекта (где находится pom.xml)
3. Собрать проект с помощью Maven:

mvn spring-boot:run

Или запустить сгенерированный jar:
java -jar .\target\delete-after-0.0.1-SNAPSHOT.jar

После запуска открыть Swagger UI в браузере по адресу:
http://localhost:8080/swagger-ui/index.html

В Swagger UI найти эндпоинт POST /excel/{n}, выбрать параметр file, загрузить Excel файл, 
указать число {n} в пути и выполнить запрос. 
Ответом будет список из N минимальных чисел из файла.
